## Tinnews, Android mobile news feed app with MVVM pattern
[project demo](https://www.dropbox.com/s/6kjnlb7xrtourhq/TinNews_compressed.mp4?dl=0)

###### Features and Components
* Implemented bottom bar and page navigation using JetPack Navigation Component
* Supported swipe gesture for like and dislike using CardStackView
* Accomplished local cache and offline model using Room Database, LiveData, and ViewModel
* Integrated Retrofit to pull latest news with RESTful API from NewAPI
